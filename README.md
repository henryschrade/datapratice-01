# datapratice-01
for search/sort practice 

Tasks
1. Display the data

Print every student and score in the original order.

2. Find the highest score

Print:

the highest score
the student's name

If multiple students share the highest score, print the first one encountered.

3. Find the lowest score

Print:

lowest score
student's name
4. Calculate statistics

Print:

average score
median score
range (highest − lowest)

Round the average to two decimal places.

5. Search by name

Prompt the user for a student's name.

If found, print their score.

If not found, print

Student not found.

Use a linear search.

6. Search by score

Prompt the user for a score.

Print every student with that score.

Example:

94

Liam
Sophia
7. Sort by score

Sort the data from highest score to lowest score.

Do not use any built-in sorting methods.

You may choose:

Selection Sort
Insertion Sort
Bubble Sort

After sorting, print the sorted list.

8. Sort alphabetically

Sort the students by name (A–Z).

Again, do not use built-in sorting methods.

Print the results.

9. Binary search (after sorting)

After sorting alphabetically, prompt the user for a student's name.

Use binary search to locate the student.

Print:

Found at index X
Score: XX

or

Student not found.
10. Challenge

Without using any additional collections:

Print the top five students in descending score order.

Example:

1. Benjamin - 99
2. Elijah - 95
3. Liam - 94
4. Sophia - 94
5. Charlotte - 91
